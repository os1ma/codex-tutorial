# リポジトリ概要

このリポジトリはシンプルな Todo CLI のサンプルコードです。実装コードは `todo/` パッケージにまとまっており、`__main__.py` から `main()` を呼び出す構成になっています。CLI の中心となる `todo/cli.py` では、以下のサブコマンドを提供しています。

- `add`: 新しいタスクを追加
- `list`: タスク一覧を表示
- `done`: 指定したタスクを完了にマーク
- `delete`: 指定したタスクを削除

タスクデータはカレントディレクトリの `tasks.json` に保存されます。依存ライブラリは特に無く、標準ライブラリのみで動作します。

## 設定とツール

- 型チェックに **mypy** の strict モードを利用しています。
- Linter として **ruff** が設定されています。
- 依存関係の管理には `uv` を想定した `uv.lock` と `Makefile` が用意されています。

## 今後の学習指針

- サブコマンドの追加方法を学ぶために `todo/cli.py` を読んでみましょう。
- 型ヒントを充実させつつ、pytest などでテストを追加してみると良いでしょう。
- `pyproject.toml` を参考に、パッケージ配布や依存管理方法も確認してみてください。
